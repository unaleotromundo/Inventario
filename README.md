# Buscador Masivo de Google Fotos

Proyecto de desarrollo web para buscar imágenes en tiempo real y con calidad original de forma simultánea en múltiples cuentas de Gmail, utilizando **Supabase** como backend y la **Google Photos Library API**.

## Configuración rápida
1. Clonar el repositorio.
2. Reemplazar las constantes `SUPABASE_URL` y `SUPABASE_ANON_KEY` en `index.html` con las credenciales de tu panel de Supabase.
3. Levantar con un servidor local (ej: Live Server en VS Code).
